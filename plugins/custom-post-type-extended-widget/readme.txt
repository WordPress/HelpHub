== Description ==

This will list and link Custom Post Type by Category inside a widget. Option to querying posts by different custom post type and its categories. Limit list to display a custom number of items. Sort ascending and descending.

== Installation ==

Installing "Custom Post Type Extended Widget" can be done either by searching for "Custom Post Type Extended Widget" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
2. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Assign your widget to a sidebar position
5. Customize output using this widget's settings
